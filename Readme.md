## Installation

Required ENV Variables

Either
-> BITBUCKET_AUTH_TOKEN"

OR
-> BITBUCKET_APP_PASSWORD
-> BITBUCKET_APP_USERNAME

Run the following command to install:

```bash
curl -fsSL https://raw.githubusercontent.com/srijan619/bbpr-terminal/main/install.sh | bash

```

Following permissions needed in the API Token:

1. Read/Write Pull Requests 
2. Read User Info (Used for filtering and customization)
